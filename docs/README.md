## Getting Started

```bash
# clone the project
git clone https://github.com/mapgis/mapgis-pan-spatial-map.git

# enter the project directory
cd mapgis-pan-spatial-map

# install dependency
yarn # Or npm install

# develop
yarn serve # OR npm run serve
```
