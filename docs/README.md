## Getting Started

```bash
# clone the project
git clone https://github.com/MapGIS/MapGIS-Pan-Spatial-Map.git

# enter the project directory
cd MapGIS-Pan-Spatial-Map

# install dependency
npm install # Or yarn

# develop
npm run dev # OR yarn dev
npm run dev:mock # OR yarn dev:mock
```
